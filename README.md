# Student Managent System

# Description

this is a simple program It manages students

Each student has
* ID
* Name
* Age
* Couser
* Aveage
* Status (active or inactive)

# How it works
the program uses
* A list --> to store students
* A dictionary --> for each student
* Functions --> to do actions
* Boolean --> to control menu

# How to run
1. Open terminal or VS code
2. create file:
   students.py
3. Run:
   python students.py

# Menu
you will se
1. Add
2. Show
3. Search
4. Update
5. Delete
6. Save CSV
7. Save TXT
8. Exit
